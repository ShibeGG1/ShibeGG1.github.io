OS install feature which limited the ram to 256MB. Its a pretty cool machine, i think i prefer it on XP even though it runs Windows 7 flawlessly.
I also have another hard drive which is 1TB and i frequently swap the XP hard drive around with that one which has Ubuntu 24.04 LTS on it
